# Сказки вселенной ИИ

Волшебные истории, созданные вместе с искусственным интеллектом.

## О проекте

Интерактивное приложение для создания и чтения сказок с помощью AI. Выбери мир — и история начнется!

### Миры

- **Русская сказка** (3-7 лет) — Жар-птицы, добрые молодцы и тридевятые царства
- **Большое приключение** (8-12 лет) — Карты, квесты и выборы с настоящими последствиями
- **Магический портал** (13+ лет) — Тайные миры, древние силы и магия

## Технологии

- React + TypeScript
- Vite
- SQLite (better-sqlite3)
- AI: Google Gemini 2.5 Flash (via N1N API)
- Framer Motion (анимации)

## Запуск

```bash
npm install
npm run dev
```

Открой http://localhost:3001

## Создание истории

1. Выбери мир на главном экране
2. Нажми "Создать сказку"
3. AI сгенерирует мир, героя и антагониста
4. Начни читать и выбирай продолжение
